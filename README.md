# Pacman Game

A desktop Pacman game built in Java with Swing. The project includes multiple game modes, ghost behavior, score tracking, sound effects, and sprite-based visuals.

## Features

- Classic Pacman-style maze gameplay
- Multiple game modes: `NORMAL`, `SPEED`, and `HARDCORE`
- Animated Pacman movement and ghost states
- Sound effects and background music
- Bonus fruit, power food, lives, and score tracking

## Project Structure

- `src/`: Java source files and game assets
- `bin/`: compiled output generated by VS Code or the Java compiler
- `lib/`: external libraries if you add any later

## Run The Game

### In VS Code

1. Install the Java extensions if prompted.
2. Open the project folder.
3. Run `App.java`.

### From the terminal

Compile:

```powershell
javac -d bin src\*.java
```

Run:

```powershell
java -cp "bin;src" App
```

## Notes

- The game is designed as a simple Java Swing project with no build tool required.
- Assets such as images and audio are stored alongside the source files in `src/`.
